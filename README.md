# stock-ai-assistant

A 股智能分析助手主项目。

本仓库是当前多人使用 Codex 协作开发的唯一主仓库。后续功能、文档、任务拆分、PR 和部署都应基于本仓库继续推进。

## 当前定位

- 产品方向：A 股智能分析助手
- 协作方式：多人开发者 + Codex + GitHub
- 主要能力：行情理解、个股分析、预测记录、风险提示、复盘沉淀
- 部署方式：见 `DEPLOY.md`

## 开发入口

开始任何开发任务前，先阅读：

1. `AGENTS.md`
2. `CONTRIBUTING.md`
3. `docs/Codex多人开发规范.md`
4. `docs/MVP方案.md`

## 项目边界

- 不承诺收益
- 不提供确定性荐股结论
- 不接真实交易下单
- 涉及行情、预测、策略、风险的输出必须保留风险提示和数据来源说明
